# 일본어 공부 앱

- url: https://study-japanese.vercel.app
- pwa로 만들었음.
- next.js + ts 프로젝트

## 사용한 스택
- Next.js
- Styled-Components
- next-pwa

## 해결해야할 것

1. 모든 글자는 100번 이내에 최소 2번은 나와야함
2. 3초 이내 맞추지 못하면 넘어감
3. 문자표 넣기 (~~히라가나~~ / ~~가타카나~~ / 한문)
4. 학습 순서

- 히라가나 => 가타카나 => 히라가나만 있는 단어 => 가타카나만 있는 단어 => 히라 + 가타 단어
  => 한자 => 문장

5. 그날의 학습률
   맞춘 문제 수/각 챕터별 푼 문제 수 퍼센트
6. 단어 수집 어떻게 하지..?
